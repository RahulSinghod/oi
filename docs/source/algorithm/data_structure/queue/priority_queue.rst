Priority queue
===============

Prirority queue (우선순위 큐)는 데이터들이 우선 순위를 가지고 있고 우선 순위가 높은 데이터가 먼저 나가는 구조


======================
Implementation methods
======================

* Array
* Linked list
* `Heap <https://oi.readthedocs.io/en/latest/algorithms/data_structure/queue/heap.html>`_
