================
Time complexity
================

**Best case**
    * The number of comparisons
        * O(n)
    * Best T(n) = O(n)


**Worst case**
    * The number of comparisons
        * O(n^2)
    * The number of exchanges
        * O(n^2)
    * Worst T(n) = O(n^2)
