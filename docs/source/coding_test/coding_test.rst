===========
코딩 테스트
===========

취업 또는 이직을 위해 통과해야 되는 기본 관문 중 하나가 코딩테스트다. 별 거 아니라고 생각할 수도 있지만 생각보다 준비를 조금 해야 한다.

여기에서는 각 기업별 코딩테스트 조사한 내용과 백준, 프로그래머스에서 문제 풀이한 내용들을 정리하려고 한다.

================================

코딩테스트 고득점 Kit
********************

위 링크에는 풀이한 모든 문제들이 정리되어 있지만, 여기에서는 프로그래머스에서 제공하는 코딩테스트 고득점 Kit 문제들만 정리하려고 한다.

문제 유형은 해식, 스택/큐, 힙, 정렬, 완전탐색, 탐욕법, 동적계획법, 깊이/너비 우선 탐색, 이분탐색, 그래프으로 총 10가지 유형에 대한 문제들을 제공하고 있다.

각 문제에 대한 풀이 링크는 다음 목록과 같다.

* 해시

* 스택/큐

* 힙

* 정렬

    * `가장 큰 수 <https://github.com/hwkim89/programming/blob/master/programmers/algo/sort/the_biggest_number.ipynb>`_

* 완전탐색

* 탐욕법

    * `큰 수 만들기 <https://github.com/hwkim89/programming/blob/master/programmers/algo/greedy/creating_big_number.ipynb>`_
    * `단속 카메라 <https://github.com/hwkim89/programming/blob/master/programmers/algo/greedy/speed_camera.ipynb>

* 동적계획법

    * `N으로 표현 <https://github.com/hwkim89/programming/blob/master/programmers/algo/dp/n_representation.ipynb>`_
    * `등굣길 <https://github.com/hwkim89/programming/blob/master/programmers/algo/dp/way_to_school.ipynb>`_

* 깊이/너비 우선 탐색

    * `단어 변환 <https://github.com/hwkim89/programming/blob/master/programmers/algo/bfs_dfs/word_change.ipynb>`_

* 이분탐색

    * `징검다리 <https://github.com/hwkim89/programming/blob/master/algo/search/binary_search/stepping_stone.ipynb>`_

* 그래프

    * `가장 먼 노드 <https://github.com/hwkim89/programming/blob/master/programmers/ds/graph/furthest_node.ipynb>`_
