==============
Neural Network
==============

위키피디아에 의하면 인공 신경망 (Artificial neural network, ANN)은 기계학습과 인지과학에서 생물학의 신경망 (Biological neural network, 동물의 중추신경계중 특히 뇌)에서 영감을 얻은 통계학적 학습 알고리즘이라고 한다. 보통 인공 신경망을 줄여서 신경망이라고 많이 부른다.

우선 정의를 해두고 나머지 많은 내용들은 필요에 따라 하나씩 정리하려고 한다. 그럼 하나씩 정복해보자!!
