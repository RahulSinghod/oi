Tensorflow
============

=============
Installation
=============

**Tensorflow, Tensorflow GPU를 동시에 설치하는 경우**

설치 순서에 따라 CPU 또는 GPU를 사용할지 결정된다.

첫번째 순서로 하면 CPU 버전이 동작하고, 두번째 순서로 하면 GPU 버전이 동작한다.

::

    sudo pip install tensorflow-gpu
    sudo pip install tensorflow

    sudo pip install tensorflow
    sudo pip install tensorflow-gpu
