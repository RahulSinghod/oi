lxml
=====

=============
Installation
=============

::

    sudo apt-get install libxml2-dev libxslt-dev python-dev
    sudo pip install lxml

=======
Errors
=======

**ImportError: cannot import name 'etree'**

lxml 버전 업그레이드 필요

::

    sudo pip install lxml --upgrade
