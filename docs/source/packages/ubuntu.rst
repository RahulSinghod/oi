Ubuntu
=======

=========
Commands
=========

**dpkg**

특정 패키지 설치여부 확인

::

    dpkg -l | grep python


**ufw**

방화벽 상태 확인

::

    ufw status verbose
    ufw status numbered


**alias**

별명 설정

::

    alias ls='ll -s'


**Disk 관련 내용**

용량 확인

::

    df -h


Mount 여부 확인

::

    lsblk


**Process 사용자 확인**

::

    ps -u -p 43401
