====
Git
====

Git은 무료로 사용할 수 있고 다양한 프로젝트에서 빠르고 효율적으로 사용할 수 있는 오픈 소스 버전 관리 시스템 (Version Control System)이다. Git과 같은 버전 관리 시스템의 목적은 버전을 관리하여 백업, 복구, 협업을 가능하게 만드는 것이다. 지금부터 아래의 목차 내용을 하나씩 살펴보자.
