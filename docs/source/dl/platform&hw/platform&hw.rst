==========================
딥러닝을 활용하기 위한 환경
==========================

딥러닝을 활용하기 위해서는 딥러닝 모델을 학습시키거나 테스트 할 수 있는 환경이 구축되어 있어야 한다. 기본적으로 Local에 딥러닝 환경을 구축할 수도 있고, Cloud에서 딥러닝 환경을 제공하는 AWS, GCP, NNSML과 같은 다양한 Platform들을 사용할 수도 있다.

여기에서는 GPU에 대한 기본내용, Local에 딥러닝 환경 구축하는 방법 (Ubuntu, Windows), Cloud 서비스들에 대해 살펴보려고 한다.
