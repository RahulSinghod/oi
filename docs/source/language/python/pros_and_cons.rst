==============
Pros and cons
==============

장점
====

* 문법이 간단하고 읽고 쓰기 쉬움
* 유용한 라이브러리가 많음
* 동적타입으로 유연함 (`덕 타이핑 <https://ko.wikipedia.org/wiki/%EB%8D%95_%ED%83%80%EC%9D%B4%ED%95%91>`_)


단점
====

* 인터프리터 언어라 속도가 느림
* 동적타입으로 자료형과 관련된 오류 발생 가능성 ↑
