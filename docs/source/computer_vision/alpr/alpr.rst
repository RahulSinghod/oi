=====
ALPR
=====

ALPR (Automatic License Plate Recognition)은 자동차의 번호판을 자동으로 인식하는 방법이다. 여기에서는 ALPR과 관련된 논문들을 하나씩 정리해보려고 한다. 그리고 다양한 연구에서 여러 가지 Dataset을 활용했는데, :doc:`여기 <dataset/dataset>` 에 정리되어 있다.
