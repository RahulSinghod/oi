=====
인적성
=====

인성 시험은 개인의 인성을 파악하기 위한 시험으로 기업이 원하는 인재상에 적절한 인성에 가까우면서 솔직하게 (?) 문제를 풀면 일반적으로 통과할 수 있다.

적성 시험은 기업에 입사전 개인의 기본적 역량을 파악하기 위한 시험으로 크게 4가지 유형으로 나뉜다.
