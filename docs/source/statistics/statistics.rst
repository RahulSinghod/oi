===========
Statistics
===========

통계학 (Statistics)은 수량적 비교를 기초로 하여, 많은 사실을 통계적으로 관찰하고 처리하는 방법을 연구하는 학문이다.
크게 기술 통계 (Descriptive statistics)와 추론 통계 (Inference statistics)로 구분할 수 있다.
